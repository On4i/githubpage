Diplomado en Bases de Datos

El "Diplomado en Bases de Datos" Proporcionar a los estudiantes de conocimientos avanzados sobre el diseño, administración y optimización de bases de datos, incluyendo tanto sistemas relacionales como NoSQL.

Duración: 448 horas académicas

Fundamentos de Bases de Datos Relacionales
  • Fundamentos de las bases de datos relacionales
  • conceptos modelos entidad-relación, normalización, álgebra relacional y lenguaje SQL
  • Almacenamiento y análisis de datos
  • Administración y seguridad en bases de datos

SQL Avanzado y Optimización de Consultas
  • Optimización de planes de ejecución
  • Uso eficiente de índices
  • Gestión de transacciones y concurrencia
  • Subconsultas, CTEs, vistas materializadas y particionamiento

Bases de Datos NoSQL
  • Tipos de bases de datos NoSQL
  • Arquitecturas y mecanismos de consulta
  • Diseño, consistencia y particionamiento en entornos distribuidos

Administración de Bases de Datos
  • Arquitectura y modelos de DBMS
  • Seguridad, rendimiento y recuperación ante fallos
  • Administración en la nube y tecnologías emergentes

DataOps y DevOps para Bases de Datos
  • Principios de DataOps
  • Prácticas de DevOps en bases de datos
  • Automatización, integración y despliegue continuo

Bases de Datos Graph
  • Estructura y operaciones de bases de datos grafos
  • Algoritmos de grafos y consultas con Cypher/Gremlin
  • Aplicaciones en redes sociales, análisis de relaciones y recomendadores

Gobernanza y Calidad de Datos
  • Marcos y estándares de gobernanza
  • Gestión y evaluación de calidad de datos
  • Políticas de seguridad, privacidad y compliance

Taller de Preparación y Publicación de Papers
  • Estructura y componentes de un paper científico
  • Métodos de investigación cuantitativos y cualitativos
  • Proceso de revisión por pares y estrategias de publicación
  • Aspectos éticos en la investigación

Sobre Nosotros

En nuestra universidad buscamos brindar a los estudiantes los conocimientos y habilidades para la construcción de aplicaciones avanzadas, utilizando tecnologías, lenguajes, enfoques y metodologías vigentes.
